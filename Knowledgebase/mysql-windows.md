# Installing a MYSQL Server (Windows

This guide explains how to install a MariaDB server directly from the official MariaDB downloads page. This does not use XAMPP or any bundled software.

## 1. Download MariaDB

1. Go to the MariaDB download page:  
   https://mariadb.com/downloads/
2. Select your operating system (Windows, Linux, or macOS).
3. Choose the MariaDB Server version you want.

## 2. Installation

1. Run the installer you downloaded.
2. Accept the license agreement.
3. Select the components you want installed. The default options are fine for most setups.
4. Set the root password when prompted. This password will be required to access the server.
5. Choose whether the service should start automatically on boot.
6. Finish the setup and let the installer start the MariaDB service.
